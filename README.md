# Algoritmo de Dijkstra – Estrutura de Dados

## Descrição
Este repositório contém a implementação do algoritmo de Dijkstra utilizando um grafo baseado em lista de adjacência.

A implementação calcula o menor caminho a partir de um vértice inicial em grafos com pesos não negativos.

## Tecnologia
Python 3.8+

## Como executar

```bash
cd src
python main.py
